## Self Host 1pt
Self-hosting 1pt requires atleast `PHP 5` and a `mysqli` database.

### Sample `mysqli` Databse Structure

| variable | type | default |
| --- | --- | --- |
| index | int | AUTO_INCREMENT
| timestamp | timestamp | current_timestamp() 
| short_url | text | 
| long_url | text | 
| desktop | text | 
| mobile | text | 
| apple | text | 
| android | text | 
| custom | text |
